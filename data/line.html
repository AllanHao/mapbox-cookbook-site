# 线

线可以用 line 表示。

# line

部分 paint 属性：

属性 描述
line-color 颜色，默认"#000000"
line-opacity 不透明度[0, 1]，默认1
line-width 线宽，默认1
line-blur 模糊度，默认0
line-dasharray 虚线模式，定义虚线和空隙长度
line-gap-width 线条空隙宽度，可实现双线

部分 layout 属性：

属性 描述
line-cap 线条末端样式，可选"butt", "round", "square"，默认"butt"
line-join 线条连接样式，可选"bevel", "round", "miter"，默认"miter"
